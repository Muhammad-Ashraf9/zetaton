# Zetaton

## How to run the project

1. Clone the repository

```bash
git clone https://github.com/Muhammad-Ashraf9/zetaton.git
```

2. Install dependencies

```bash
cd zetaton
npm install
```

3. Run the project

```bash
npm start
```
